# Кухонный калькулятор v1.0

Кухонный калькулятор — это консольная программа для Windows, которая помогает считать КБЖУ продуктов и рецептов.

## Что умеет программа

- считать калории, белки, жиры и углеводы
- искать рецепты
- показывать список рецептов
- вести историю расчётов
- работать с избранным
- добавлять свои блюда
- добавлять свои продукты
- работать с водным счётчиком
- очищать тестовые данные

## Как запустить

1. Скачайте ZIP-файл из релиза.
2. Распакуйте архив.
3. Запустите файл `Recepts.exe`.
4. Введите название рецепта или команду.
5. Для выхода напишите: `выход`.
